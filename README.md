# Options Pricing — Black-Scholes & Monte Carlo

European Call/Put pricing with Greeks calculation and dynamic delta hedging.

## Key Results
- BS vs MC price convergence across 10,000 GBM paths
- Greeks: Delta, Gamma, Theta, Vega, Rho
- Delta hedging P&L convergence visualization

## Tech Stack
Python · SciPy · NumPy · Matplotlib

## Run
pip install -r requirements.txt
python options_pricing.py
